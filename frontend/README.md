# 🌐 Frontend – README

## 📦 Requirements

* Node.js 22+
* npm / yarn / pnpm

## ⚙️ Installation & Setup

```bash
cd frontend
npm install
```

## 🔑 Environment Variables

Create a `.env` file in `frontend/` with:

```env
VITE_BACKEND_URL=http://127.0.0.1:5000
VITE_CLERK_PUBLISHABLE_KEY=your_clerk_publishable_key_here
```

## ▶️ Run

```bash
npm run dev
```
